# Simple Interest Calculator

This project is a simple web-based calculator used to calculate Simple Interest.

## Formula
Simple Interest = (P × R × T) / 100

Where:
- P = Principal Amount
- R = Rate of Interest
- T = Time Period

## Technologies Used
- HTML
- CSS
- JavaScript

## Features
- User-friendly interface
- Instant calculation
- Responsive design
